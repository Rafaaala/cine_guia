# 🎬 Guia de Cinema

Aplicação web desenvolvida com HTML5 e CSS3, focada em acessibilidade, responsividade e experiência do usuário para consulta de filmes.

## ✨ Funcionalidades

- Catálogo de filmes com informações e avaliações
- Filtros de gênero interativos (sem JavaScript)
- Layout responsivo para desktop e mobile
- Navegação por teclado (acessibilidade)
- Interface moderna e contrastante
- Animações e efeitos visuais com CSS

## 🧩 Tecnologias

- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- Google Material Symbols

## 🎯 Objetivo

Oferecer um guia visual de filmes, permitindo ao usuário filtrar por gênero, visualizar detalhes e navegar de forma acessível em qualquer dispositivo.

## 📊 Acessibilidade

- Labels e navegação por tabulação
- Estrutura semântica e headings claros
- Contraste de cores adequado
- Compatível com ferramentas como Lighthouse
